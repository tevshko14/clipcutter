# ✂ ClipCutter

## Install (one time)

1. Put both files (`clipcutter.py` and `Install_ClipCutter.command`) in the same folder
2. Open **Terminal** (Cmd+Space, type "Terminal")
3. Paste this single line — replace the path with where you saved the files:

```
bash ~/Downloads/Install_ClipCutter.command
```

4. Wait a few minutes — it installs everything automatically
5. **ClipCutter.app** appears on your Desktop

## Use

Double-click **ClipCutter.app**. That's it.

First time macOS may block it — right-click → Open → Open.
